Linear Regression¶

  • A model that represent the linear relation between a dependent variable "y" and one or more independing variables "X".

  • The model is a line the line equation is shown below $ y = slope * X + bias $

  • We will create synthetic data set, then add some noise to it and use tensorflow to create a model that predicts the correct labels of the data using its noise counterpart.

TensorFlow and Linear Regression¶

Using tensorflow we will find the optimal weigths for the noise data which will be as close as possible as the real weigths. Tensor flow is finding a linear function that represents the original data .

# INPUT features.iloc[:, :3]
# LABEL features['noise_labels']
# PARAMETERS w_real, b_real
# LABEL prediction Linear Regression
# LOSS function R^2 mean square differece
# OPTIOMIZATION function Gradient Descent

# Number of steps.
# On each epoch the weigths are optimize
NUM_EPOCHS = 200

# Create a non-defualt graph
g = tf.Graph()

# Set a list to hold the regression parameters
weigth_bias_parms = []

# Graph
with g.as_default():
    x = tf.placeholder(tf.float32,shape=[None,3])
    y_true = tf.placeholder(tf.float32,shape=None)
    
    # Define scopes per each stage of the main session
    
    with tf.name_scope('inference') as scope:
        # Create tensorflow weigths variable shape,type,name
        w = tf.Variable([[0,0,0]],dtype=tf.float32, name='weights')
        # Create tensorflow bais variable shape,type,name
        b = tf.Variable(0, dtype=tf.float32, name='bias')
        # Generate the prediction
        y_pred = tf.matmul(w,tf.transpose(x)) + b
   
    with tf.name_scope('loss') as scope:
        loss = tf.reduce_mean(tf.square(y_pred - y_true))
        
    with tf.name_scope('train') as scope:
        learning_rate = 0.5
        optimizer = tf.train.GradientDescentOptimizer(learning_rate)
        train = optimizer.minimize(loss)
        
    # Initialize the Variables by creating its content
    init = tf.global_variables_initializer()
    
    # Start the session
    with tf.Session() as sess:
        sess.run(init)
        print("Iniatl weigths and bias")
        print(sess.run([w,b]),"\n")
        print("Optimized weigths...")
        for epoch in range(NUM_EPOCHS):
            sess.run(train,feed_dict={x: features.iloc[:, :3], y_true: features['noise_labels']})
            # Every 5 epochs print the parameters
            # append this to a list
            if (epoch % 50 == 0):
                print(epoch, sess.run([w,b]))
                weigth_bias_parms.append(sess.run([w,b]))
        
        # Print the optimized weights and bias
        print(200,sess.run([w,b]),"\n")
        
        # Print Original weigths
        print("Original Weigths\n")
        print(w_real, b_real)

Note that there is a second slope factor multiplying the Vibration parameter which
it is not shown here; this is important to define the boundary equation. The slope
and the Intercept will be replaced by the parameters w and b.

The decision boundary equation is equal to¶

$intercept + slope * RPM = Vibration$
or
The boundary is a hyperplane so its equation can be re-written as:
$intercept + slope * RPM - Vibration = 0$
or
$b + w_1*RPM - w_2*Vibration = 0$
or
$b + w_1*x_1 - w_2*x_2 = 0$
Anything above the decision boundary is "-" and anything below is "+"
